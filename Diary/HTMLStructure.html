The structure of HTML files

In general an HTML document describes the structure and content of a web page. It can include text, images, links, and other elements.
An HTML element is defined by a start tag, some content and an end tag.
Some HTML elements have no content (like the br element). These elements are called empty elements. Empty elements do not have an end tag.
An HTML element can also have attributes, which are additional information about the element. Attributes are specified in the start tag and usually have a name and a value.
HTML documents are displayed in web browsers, which read the HTML code and render it according to the specified structure and style. Web browsers do not display the HTML tags, but use them to determine how to display the document.
If you open the website in the browser, the browser creates a document object model of the page.

!DOCTYPE html

This declaration defines that this document is an html document. It's a required preamble to make sure the doc behaves correctly.

html

This is the root element of an html page. It wraps the content of the entire pages.

head

This element contains meta information about the HTML page. It incorporates all the stuff you want to include on the html page, but isn't the content the viewers gonna see.

title

This element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab).

body

This element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

h1

This element defines a large heading.

head

This element defines a paragraph.
