The basics of CSS

Syntax

  • Apart from the selector, each ruleset must be wrapped in curly braces. ({})
  • Within each declaration, you must use a colon (:) to separate the property from its value or values.
  • Within each ruleset, you must use a semicolon (;) to separate each declaration from the next one.
  • You can also select multiple elements and apply a single ruleset to all of them. Separate multiple selectors by commas.

Selectors

A CSS selector is the first part of a CSS Rule.
It is a pattern of elements and other terms that tell the browser which HTML elements should be selected to have the CSS property values inside the rule applied to them.
The element or elements which are selected by the selector are referred to as the subject of the selector.


Selector lists

If you have more than one elementusing the same selector you can combine them in a list. You can do this by adding a comma between them (white space is valid before or after the comma).


DIfferent Types of Selectors

  • Type Selector: it targets an html element, e.g. my-element
  • Class Selector: it targets an element with a specified class, e.g. .my-class
  • ID Selector: it targets an elementwith a certain id, e.g. #my-id
  • Attribute Selector: it targets elements with specified attributes
  • Pseudo-class Selector: it only targets the elements when they are in a specified state

CSS Boxes

CSS is based on the box model system.
Each box placed on your page is taking up sapce and has properties like:


  • padding: the space around the content
  • border: the line that is just outside the padding
  • margin: the space around the outside of the border
  • CSS knows the directions in the order: above, right, under, left
  • others: universal selector (asteriks), hierarchy
